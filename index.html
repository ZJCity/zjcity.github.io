Stomper: A 2D Platformer (University)

My Java project is a 2D platformer game called Stomper. It is a game where you stomp on the heads of enemies and collect gems. This project allowed me to explore Java and it's concepts. I learned a lot about documentation through JavaDoc. One of the main challenges that I encountered was detecting collisions on entities that were loaded through a save file. I was able to fix this by using global collision handlers that the level would load.
